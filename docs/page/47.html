Olá Homelabers! Notícia fresquinha: A VMware acabou de anunciar as novas versões do VMware Workstation Player 12.5 e VMware Workstation Pro 12.5. As atualizações são gratuitas para usuários da versão 12. Esta versão suporta o Windows 10 Anniversary Edition da Microsoft, e está pronto para suportar o Windows Server 2016 após a sua libertação.
